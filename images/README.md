# Blog Images

Place your blog post images here.

## Usage in Posts

```markdown
![Alt text]({{ site.baseurl }}/images/your-image.png)
```

## Organization

You can organize in subdirectories:
```
images/
  ├── 2025-11-04-post-name/
  │   └── figure.png
  └── shared/
      └── logo.png
```

Reference: `![]({{ site.baseurl }}/images/2025-11-04-post-name/figure.png)`
